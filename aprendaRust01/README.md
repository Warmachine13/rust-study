# rust-study

# aprendaRust01

To build project aprendaRust01 you need to run command

```sh
rustc hello.rs
```

and to run 

```sh
./hello.rs
```

# aprendaRust01 cargo

To build *debug project aprendaRust01 you need to run command

```sh
cargo build
```

and to run 

```sh
./target/debug/proj
```

to make relase

```sh
cargo build --release
```

and to run 

```sh
./target/release/proj
```

and to automate cargo 

```sh
cargo run
```

to lint project 


```sh
cargo fmt
```
